Slime

  • In your .emacs file, point your Lisp implementation to the eql executable, adding the command line option -slime: 
      (setq inferior-lisp-program "/path-to-eql/eql -slime")
  • For help in Slime mode, see the qapropos and qgui functions.
  • Qt event processing is near-to-native, see eql/slime/process-qt-events.lisp.
