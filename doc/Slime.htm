Slime

This should work on any platform, with any Slime version that plays together with ECL.
Requires ECL threads.

    Prepare

    • Add to your ~/.emacs file: 
      (add-to-list 'load-path "~/slime/")         ; slime path
(add-to-list 'load-path "~/slime/contrib/") ; slime/contrib path
(require 'slime)
(slime-require 'swank-listener-hooks)       ; EQL requires a listener hook
(slime-setup '(slime-fancy))
    • Add to your ~/.swank.lisp file (or copy file eql/slime/.swank.lisp in your home directory): 
      (setf swank:*globally-redirect-io* t) ; show print output in Emacs
    • Copy the file eql/slime/start-swank.lisp in your slime/ directory

    Run

    • Run the swank server: 
      eql -slime ~/slime/start-swank.lisp
    • Fire up Emacs and do: Meta-X slime-connect (important: use slime-connect) and hit Return 2 times

    Notes

    If all went well, you should have a Slime REPL processing Qt events.

    For help see the qapropos and qgui functions.
