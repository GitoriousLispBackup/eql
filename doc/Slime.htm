Slime

  • Please get this version from CVS (as later versions don't currently work with the integration provided by EQL):
    Unix, OSX: cvs -d :pserver:anonymous:anonymous@common-lisp.net:/project/slime/cvsroot co -D 2011-11-01 slime
    Windows: cvs -d :pserver:anonymous:anonymous@common-lisp.net:/project/slime/cvsroot co -D 2010-06-01 slime
    On Windows my experience is a bit frustrating: the only working solution for me is:
    ECL 11.1.1 + Slime 2010-06-01 + Windows XP (MSVC)
  • In your .emacs file, point your Lisp implementation to the eql executable, adding the command line option -slime: 
      (setq inferior-lisp-program "/path-to-eql/eql -slime")
  • For help in Slime mode, see the qapropos and qgui functions.
  • Qt event processing is near-to-native, see eql/slime/process-qt-events.lisp.
